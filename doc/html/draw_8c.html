void afficher_vie_joueurs ()
 Les personnages ont une bare de vie, donc à tout moment le joueur est ammené à savoir combien de vie a son personnage, cette fonction affiche à l'écran la vie de tous les joueurs.
 
void afficher_ecran_multijoueur ()

void afficher_pause ()

void modifier_window ()
